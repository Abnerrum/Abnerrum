# Projeto com Python + SQLite

Projeto simples de **CRUD de tarefas** usando apenas biblioteca padrão do Python e banco local SQLite.

## Requisitos

- Python 3.10+

## Como executar

```bash
cd python_sqlite_projeto
python app.py
```

Na primeira execução o arquivo `projeto.db` e a tabela `tarefas` são criados automaticamente.

## Funcionalidades

- Criar tarefa
- Listar tarefas
- Marcar tarefa como concluída
- Excluir tarefa
